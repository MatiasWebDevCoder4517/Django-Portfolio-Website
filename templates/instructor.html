Description:

Instructor Course web application with Django that allows users to create, edit, and delete profile. Instructors can manage full courses with featured attachments to advance the knowledge of theirs enrolled students, like tutorials, file documents , announcements and other stuff. Some of the functionalities are:

CRUD of users, tutorials, courses, etc
Smart user/post search bar.
Instructors can 'CRUDs' pdfs, videos, announcements, etc
User control and ability to add user profile. User Security encryption and data management
User can enroll in any tutorial, course or download study material.
Comments and post functionalities.
Multiple validations and error flow control.
Responsive Implementation for mobile users.
and many other features...

Technologies Used:
  • - Python
  • - Django
  • - HTML/CSS/JS
  • - BOOTSTRAP
  • - POSTGRESQL
  • - AWS Deployment
